# Claude Code 源码学习 · 进度跟踪

> 学习的根本目的:**不是会用 Claude Code,而是看懂一个真正能用的 Agent 是怎么工程化落地的。**
>
> 我们以"泄漏出来的 cli.js 打包产物 + 公开文档 + 可观测行为"为参照,从理论上还原它的设计。
> 不抄代码、不背配置,关注**机制**与**取舍**。

---

## 学习路线总览

整个学习被拆成 8 个阶段、约 20 课。每一课围绕一个**核心机制**展开,而不是按照功能列表流水账。

| 阶段 | 主题 | 课次 | 状态 |
| --- | --- | --- | --- |
| 一、基础认知 | 它到底是什么 | 01 - 02 | ⏳ |
| 二、Agent 循环核心 | 主循环、工具循环、流式处理 | 03 - 05 | 待开始 |
| 三、Prompt 工程 | System Prompt、上下文管理 | 06 - 07 | 待开始 |
| 四、工具系统 | 定义、调度、典型工具 | 08 - 10 | 待开始 |
| 五、扩展机制 | Subagent、Hook、Slash、MCP | 11 - 14 | 待开始 |
| 六、权限与安全 | 权限模型、注入防御 | 15 - 16 | 待开始 |
| 七、工程实现 | 持久化、TUI、Memory | 17 - 19 | 待开始 |
| 八、综合 | 自己设计一个 Agent CLI | 20 | 待开始 |
