# deep learning 学习路径与知识图谱

## 知识图谱（从基础到进阶）

1. 预备基础
   - Python 与张量计算（NumPy/PyTorch 基础）
   - 线性代数（向量、矩阵、矩阵乘法）
   - 概率统计（分布、期望、方差）
   - 微积分直觉（导数、链式法则、梯度）
2. 深度学习核心
   - 神经网络与前向传播
   - 损失函数与反向传播
   - 优化算法（SGD、Momentum、Adam）
   - 正则化（Dropout、权重衰减、早停）
3. 主流模型
   - CNN（图像任务）
   - RNN/LSTM/GRU（序列任务）
   - Transformer（NLP 与多模态核心架构）
4. 工程实践
   - 数据清洗与划分
   - 训练监控与实验管理
   - 调参策略与错误分析
   - 模型部署与推理优化
5. 进阶专题
   - 迁移学习与微调
   - 自监督学习
   - 生成模型（VAE/GAN/扩散模型）
   - 大模型训练与对齐基础
